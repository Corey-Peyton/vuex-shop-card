# shop-card-vuex
I created this project to learn how to use vuex. The project is similar to the shopping cart in the e-commerce web application.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
